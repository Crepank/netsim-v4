# netsim

netsim is an AI-powered browser simulation for rapid web creation and prototyping. It provides a web UI, project/version management, AI-driven site generation, file uploads, social features (likes/comments), and a lightweight per-project database system.

## Features
- SPA shell with project editor, preview iframe, and sidebar for projects/versions/files/social.
- AI-driven generation using configurable models (Pollinations / OpenAI-compatible).
- Per-project multi-file virtual filesystem, versioning, pinning, and forking.
- Upload files (text → stored in project, binary → saved to uploads and referenced).
- User accounts (signup/login), avatar uploads, profile projects, and homepage configuration.
- Social features: likes and comments.
- Export project as ZIP.
- Simple SQLite-backed "databases" feature with per-db sqlite files. (buggy/broken)
